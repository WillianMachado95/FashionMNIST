<h1> Classificação de Imagens com Fashion MNIST </h1>

Este projeto apresenta a implementação de um modelo de machine learning  para classificação de imagens utilizando o dataset Fashion MNIST, amplamente adotado como benchmark em visão computacional e deep learning.

O objetivo principal é explorar o fluxo completo de um problema de classificação supervisionada, desde a preparação dos dados até o treinamento e avaliação do modelo.

<h2> Objetivos do Projeto </h2>

 * Aplicar conceitos fundamentais de Machine Learning e Deep Learning
 * Trabalhar com dados de imagens em escala de cinza
 * Desenvolver e treinar um modelo de classificação multiclasse
 * Avaliar o desempenho do modelo por meio de métricas adequadas

<h2> Aprendizados </h2>

 * Manipulação de dados de imagem
 * Construção de pipelines de Machine Learning
 * Treinamento e avaliação de modelos de classificação
 * Interpretação de métricas de desempenho
